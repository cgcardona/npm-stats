# npm-stats

Create a .csv file of daily downloads for an npm lib.

```ts
ts-node npm-stats.ts
prompt: npmPackage:  avalanche
prompt: startDate:  2020-09-21
prompt: endDate:  2021-04-19
npm-stats.csv created
```
